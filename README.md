# Analyse de la contamination au chlordécone – Martinique
Auteur : Yolan Denesles | Date : 30/03/2026

## Description
Script R Markdown d'analyse statistique et cartographique de la
contamination au chlordécone dans les sols martiniquais.
Inclut une analyse exploratoire, des tests ANOVA, une régression
linéaire multiple, un modèle Random Forest et 4 cartes thématiques.

## Données requises
- Data/BaseCLD2026.xlsx
- Shapefile Martinique (package cartography : mtq.gpkg)

## Librairies
dplyr, ggplot2, sf, cartography, tidyverse, patchwork,
readxl, randomForest, viridis, ggthemes, stringr

## Exécution
Ouvrir dans RStudio et cliquer sur "Knit" pour générer le rapport HTML.
